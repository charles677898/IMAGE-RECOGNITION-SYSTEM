# Image Recognition System

## Overview

An AI-powered Image Recognition System built using Python, Flask, TensorFlow, and SQLite. Users can upload images and receive real-time object recognition results with confidence scores.

## Features

* Upload image for prediction
* AI-based image classification using MobileNetV2
* Top 3 prediction results
* Confidence score display
* Prediction history storage
* SQLite database integration
* Analytics dashboard
* Professional responsive UI

## Technologies Used

* Flask

## Installation

1. Create virtual environment

```bash
python -m venv venv
```

2. Activate environment

```bash
venv\Scripts\activate
```

3. Install requirements

```bash
pip install -r requirements.txt
```

4. Run application

```bash
python app.py
```

5. Open browser

```text
http://127.0.0.1:5000
```
